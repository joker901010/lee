- IDE : IntelliJ
- Spring Boot 2.1.0
- Maven
- Java 8
- MySQL
- JPA

- 스프링 부트 프로젝트 생성
- Lombok 라이브러리
- 컨트롤러 테스트

## 3. demo2 내용 : 스프링부트 Data JPA

- JPA 간단 요약 정리
- 스프링부트 프로젝트 생성
- DataSource 설정
- 엔티티 클래스 설계
- Repository 인터페이스 설계
- 엔티티 테스트
- CRUD 테스트
